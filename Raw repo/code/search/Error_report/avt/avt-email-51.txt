	Title		: Extended Secure RTP Profile for RTCP-based

	Author(s)	: J. Ott, E. Carrara
	Filename	: draft-ietf-avt-profile-savpf-10.txt

	Date		: 2007-3-5
=09
An RTP profile (SAVP) is defined for secure real-time=20
   communications, and another profile (AVPF) is specified to provide=20
   timely feedback from the receivers to a sender.  This memo defines=20
   the combination of both profiles to enable secure RTP=20
   communications with feedback.
